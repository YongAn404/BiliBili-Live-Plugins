## 如何使用？

### 第一步 运行哔哩哔哩互动姫

**本项目分为两个项目文件夹**

**Live为项目源代码**

**部分用户只需要找到哔哩哔哩互动姫_1.6.8.2并解压，然后运行哔哩哔哩互动姫.exe**

### 第二步 开始使用本插件

**打开直播设定**

**房间号输入你自己的*

**设置完成后点击开始直播**
