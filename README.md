# 如何使用？

## 第一步 运行哔哩哔哩互动姫

>**作者己配置好互动姫，请勿修改乱配置**

**解压 BiliBiliInteraction_1.6.8.2_X.X.X**

>**BiliBiliInteraction_互动姫版本号_插件版本号**

**运行\install\AccessDatabaseEngine.exe并安装**

**安装AccessDatabaseEngine完成后运行哔哩哔哩互动姫.exe**

## 第二步 开始使用本插件

**打开直播设定**

**房间号输入你自己的**

**设置完成后点击开始直播**
